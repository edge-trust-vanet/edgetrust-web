## 📖 About The Project

Traditional Public Key Infrastructure (PKI) authenticates vehicles but cannot determine whether an authenticated node behaves maliciously after joining the network. **EdgeTrust-VANET** solves this by integrating **behavioral trust management** with **machine learning-based attack detection** at the network edge.

This repository (`edgetrust-web`) serves as the public-facing portfolio and documentation hub for the project. 

It features:
- 🚗 **Live Interactive Hero:** A custom HTML5 canvas simulating microscopic vehicular mobility and edge-node trust communication.
- 📚 **Research Survey:** A sleek, dark-mode compatible document viewer detailing our literature survey of 285 peer-reviewed publications.
- 📝 **Development Blog:** A timeline-based feed documenting our progress from SUMO simulations to OMNeT++ network integration.

*(Note: The actual C++ and Python simulation code resides in our core simulation repository).*

## 🚀 Getting Started

To run this website locally on your machine, follow these simple steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/edge-trust-vanet/edgetrust-web.git
   cd edgetrust-web
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the site:**
   Open your browser and navigate to `http://localhost:5173` (or the port specified in your terminal).
